# Privacy & Data Policy

## Your Data Stays Local

Wealth Tracker stores all your financial data locally on your machine in an encrypted SQLite database. **No data is sent to any server.** There is no cloud storage, no backup to external servers, and no synchronization with remote services. Your wealth data—accounts, transactions, balances, prices, and reports—never leaves your computer unless you explicitly export it.

When you create encrypted backups in Settings, those backup files are saved to your local file system and encrypted with AES-256-GCM encryption using a password you choose. You control where backups are stored and who has access to them.

## Zero Telemetry & Analytics

Wealth Tracker collects **zero analytics, zero crash reports, and zero usage data.** We do not track:
- How often you use the app
- Which features you access
- What data you enter
- System information or device details
- Browsing or location information

The application runs entirely offline. No telemetry libraries, no third-party integrations, no advertising networks. There are no external API calls except those you explicitly configure (e.g., if you manually add currency rates from external sources in the future).

## Security

- **Local encryption:** Database file on disk is unencrypted by default, readable only with app access; encrypted backups are AES-256-GCM
- **No telemetry:** Zero calls home, zero analytics
- **No account:** No login, no password to Wealth Tracker servers, no account infrastructure
- **Proprietary source rights:** Source code is protected intellectual property; reuse, modification, or redistribution requires explicit written consent

## Data Ownership

You own all data in Wealth Tracker. You can export your financial data at any time via the Reports section or by copying the database file directly. There are no restrictions, paywalls, or lock-in preventing you from accessing or migrating your data elsewhere.

## Policy Updates

This privacy policy may be updated in future versions. Any changes will be clearly noted in release notes. Material changes will only be made with your explicit consent.

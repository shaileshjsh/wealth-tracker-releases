Installation

  1. Download Wealth Tracker_1.0.0_aarch64.dmg from the latest release.
  2. Open the DMG file.
  3. Drag Wealth Tracker to your Applications folder.
  4. Launch from Applications and follow the guided setup tour.
⚠️ If macOS shows a security warning, go to System Settings → Privacy & Security and click "Open Anyway". This is expected for apps distributed outside the Mac App Store.
